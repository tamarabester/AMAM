Foobar API Reference
====================

This is Foobar's API reference.

Foo
---

.. class:: foobar.Foo

    Encapsulates ``Foo`` objects.

    .. method:: run()

        Returns ``'foo'``.

    .. method:: inc(x)

        Returns ``x`` incremented by one.

    .. method:: add(x, y)

        Returns the sum of ``x`` and ``y``.

Bar

    .. method:: add(x, y)
